## 3. 피드백 및 주요 키워드 요약

* **긍정적 피드백**:
  * 객실 위생 상태가 매우 깨끗하고 현대적임.
  * 생수 4병 기본 제공, 칫솔 및 스킨케어 어메니티 무상 제공에 대한 만족도가 높음.
  * 대형 TV와 유튜브, 넷플릭스 등 OTT 스트리밍 서비스 이용이 용이함.
  * 헤어드라이어 바람 세기가 강해 투숙 시 편리했다는 언급이 잦음.
* **아쉬운 피드백 (불만 및 개선 제안)**:
  * **체크인 대기 시간**: 밤 10시(22:00) 이후에야 체크인이 가능한 규정으로 인해 짐 보관 후 외부에서 긴 시간 대기해야 하는 불편함이 빈번히 언급됨.
  * **연박 불가 규정**: 처음에 동일한 예약으로 묶지 않고 각각 결제한 경우, 동일한 객실이라도 연박으로 인정받지 못해 정오(12:00)에 체크아웃 후 밤 10시까지 대기해야 하는 불합리한 시스템에 대한 강한 지적이 존재함.
  * **위치적 특성**: 유흥가/모텔 밀집 구역에 인접해 있어 들어갈 때 다소 불안감을 느꼈으나, 호텔 내부는 훌륭했다는 평이 다수.

## 4. 데이터베이스 및 스크립트 구조

### [1] SQLite DB 테이블 스키마 (`reviews` 테이블)
데이터베이스 파일인 `trip_com/data/reviews.db`의 구성 필드 목록입니다.

| 컬럼명 | 데이터 타입 | 설명 |
| :--- | :--- | :--- |
| **id** | INTEGER (PK) | 리뷰 고유 ID |
| **nickname** | TEXT | 작성자 닉네임 |
| **avatar_url** | TEXT | 아바타 이미지 주소 |
| **create_date** | TEXT | 리뷰 작성 일시 |
| **checkin_date**| TEXT | 체크인 일시 |
| **rating** | INTEGER | 전체 만족도 평점 |
| **rating_location** | INTEGER | 위치 만족도 평점 |
| **rating_facility** | INTEGER | 시설 만족도 평점 |
| **rating_service** | INTEGER | 서비스 만족도 평점 |
| **rating_room** | INTEGER | 객실 만족도 평점 |
| **content** | TEXT | 원본 리뷰 텍스트 |
| **language** | TEXT | 원본 리뷰 작성 언어 |
| **translated_content** | TEXT | 한국어 번역 텍스트 (있는 경우) |
| **translated_language** | TEXT | 번역 언어 (ko) |
| **room_name** | TEXT | 투숙 룸 이름 |
| **travel_type** | TEXT | 여행 타입 (예: 1인 여행자, 커플 등) |
| **useful_count**| INTEGER | 도움됨 수 |
| **recommend** | INTEGER | 추천 여부 (0: 비추천, 1: 추천) |
| **image_list** | TEXT | 이미지 URL 목록 (JSON 배열 형식) |
| **hotel_reply** | TEXT | 호텔 측 답글 내용 |
| **collected_at**| TEXT | 수집 타임스탬프 (CURRENT_TIMESTAMP) |

### [2] 스크립트 파일
* **수집기 스크립트**: `trip_com/src/scraper.py`
* **검증 및 분석 통계 출력**: `trip_com/src/verify_db.py`
